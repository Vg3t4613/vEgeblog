残念#so close

2021-02-06 3 min read # 想法

我最近是突然想起这个词，最开始接触到这个日语概念是在〇〇小日子里面，对副班长的描述
看上去很完美无缺，感觉什么都能做，都会，但事实上做什么都差一点点
我其实很久之前就这样觉得了，我确实也是这样的，很多事，但是在事后总结的时候---一开始我做事的时候是不愿意接受的，我个人能力上确实不足，如果是没有“早知道”的前提下，我能想出来的最优解也只能是我做过的那些，但事实是，确实存在更优解，在看到柯妮和刘湘衡做事之后我愈发这么觉得，我的能力上限只能支持我走到那一步。
我以前是怎么想的呢，我是一直觉得，别人能做到的，人脑构造差别不大吧，那我应该也能做得到，自己去尝试做别人做得厉害的地方，去掌握要领和理解方法，向着变“强”的方向发展自己，结果就是，变成了什么都懂一点点的人，这里懂一点那里懂一点感觉什么都被我懂完了，所以很多人一开始认识我的时候都会觉得我很高深莫测的感觉，聊相关话题深入下去我也能接一些，对一些事物能给出自己的一套看法，所以愿意跟我聊天的人也不少。麦鱼就很喜欢我这种特性，快把我当百度和知乎了，我现在还得给他兼职教练，他想玩啥我居然都能教一些，除了moba游戏。
但是也只能教一些皮毛，和游戏理解的发展方向，深入下去，我自己都没到那个水平。cs能教不少，不过大多数都是复读我教练的。

但是我也只能聊话题性的东西，聊生活日常和八卦，我就不太擅长了，现在好了点。所以我是清楚的，我的话语很多时候不太能走进别人的心中，我掏心窝说的话，大多会冷场，然后被遗忘。
不知道宣祐和刘爷是怎么做到的，他们对他们的现任说了什么，能把我想做到而做不到的时期在短时间内完成了
我不怨恨他们，我觉得他们有这个能力做到，这个成果属于他们，与我无关，无论以前发生过什么，我现在只是一个过客罢了

残念

那我后悔吗，说不后悔，心里没有想法，那肯定是假的，但是我不会说，即使是在这里，我说了，写下来，我以后看见了，原来我心里还有这样的想法，记忆的通路被恢复，又得自扰一段时间，那不值得。

我现在是已经接受我这个残念的属性，无论是打游戏也好，学习也好，人际交往也好，我就是我，我也一直在强调这一点，关于我和“我”之间我也有过一段想法，下次再写。

但无论生活中，如何不如意了，也请热爱生活
